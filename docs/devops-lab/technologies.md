# Key Technologies

The DevOps Lab combines several technologies into one coordinated local environment.

## Core stack
- Vagrant
- Oracle VirtualBox
- Kubernetes
- Docker
- Jenkins
- Ansible
- NFS
- Git and GitHub
- MySQL
- Node.js
- MetalLB
- Ingress Controller

## Why this combination matters
This mix allows users to practice not just one tool but the interaction between infrastructure provisioning, containerization, orchestration, storage, automation, CI/CD, and application exposure.
