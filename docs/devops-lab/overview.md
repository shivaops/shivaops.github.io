# DevOps Lab Overview

ShivaOps DevOps Lab is a fully automated local practice environment designed to simulate a realistic DevOps setup on a Windows laptop using Vagrant and Oracle VirtualBox.

Its main value is that it removes repetitive manual infrastructure setup and gives a ready-made platform for repeated learning, testing, and demonstration.

## Main goals
- provide a reusable local DevOps environment
- support Kubernetes, CI/CD, and automation practice
- allow browser-based app exposure through local networking tools
- create a low-risk lab without cloud cost
