# ShivaOps Docs

MkDocs + Material for MkDocs based GitHub Pages site for:

- Smart Trip – Agentic AI Travel App
- ShivaOps DevOps Lab
- Generative Report Parameter Form AI

## Run locally

```bash
pip install -r requirements.txt
mkdocs serve
```

Open `http://127.0.0.1:8000`

## Build site

```bash
mkdocs build
```

## Publish to GitHub Pages
Use this repository as the source for your `shivaops.github.io` user site or publish through GitHub Pages / GitHub Actions.
